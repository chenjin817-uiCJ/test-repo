# 家具卖点字典

一个现代化的家具材质和面料卖点展示应用，支持Excel数据导入和交互式浏览。

## 功能特性

- 🏠 **材质展示** - 精美的卡片式材质列表展示
- 🧵 **面料展示** - Gallery模式的面料型号展示，支持三大平台分类
- 🔍 **智能搜索** - 支持按材质名称、特性、卖点等关键词搜索
- 📊 **Excel导入** - 支持批量导入材质和面料数据
- 📱 **响应式设计** - 适配桌面和移动设备
- 🎨 **现代UI** - 美观的渐变设计和动画效果
- 🏷️ **平台分类** - 面料支持亚马逊、自建站、Wayfair三大平台分类

## 使用方法

### 材质管理
1. **浏览材质** - 在材质字典页面查看所有材质卡片
2. **搜索材质** - 使用顶部搜索框快速查找
3. **查看详情** - 点击材质卡片查看详细信息
4. **编辑材质** - 在详情页面点击"编辑"按钮修改材质信息
5. **导入数据** - 点击"上传Excel"按钮批量导入数据

### 面料管理
1. **浏览面料** - 切换到面料型号页面，Gallery模式展示
2. **平台分类** - 点击亚马逊、自建站、Wayfair按钮查看对应平台的面料
3. **查看详情** - 点击面料卡片查看详细特性和信息
4. **导入面料** - 点击"上传面料数据"按钮批量导入面料数据

## Excel数据格式

### 材质数据格式

请按照以下格式准备材质Excel文件：

| 列 | 内容 | 说明 |
|---|---|---|
| A | 材质名称 | 材质的名称 |
| B | 材质描述 | 材质的详细描述 |
| C | 主要特性 | 用逗号分隔的特性列表 |
| D | 图片URL | 可选，网络图片链接 |
| E | 材质类型 | 可选，wood或fabric |

### 面料数据格式

请按照以下格式准备面料Excel文件：

| 列 | 内容 | 说明 |
|---|---|---|
| A | 面料型号 | 面料的唯一型号标识 |
| B | 面料名称 | 面料的名称 |
| C | 面料特性 | 用逗号分隔的特性列表 |
| D | 图片URL | 可选，网络图片链接 |
| E | 平台分类 | amazon、website、wayfair |

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **样式**: 现代CSS Grid和Flexbox布局
- **Excel处理**: SheetJS (xlsx.js)
- **图标**: Font Awesome
- **动画**: CSS3动画和过渡效果

- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## 开发说明

### 文件结构

```
├── index.html              # 主页面
├── styles.css              # 样式文件
├── script.js               # JavaScript逻辑
├── sample_data.xlsx        # 材质示例Excel文件
├── sample_fabrics.xlsx     # 面料示例Excel文件
└── README.md              # 说明文档
```

### 核心功能

1. **数据管理** - 使用JavaScript数组管理材质和面料数据
2. **搜索过滤** - 实时搜索和过滤功能
3. **模态框** - 材质/面料详情和上传功能
4. **编辑功能** - 支持编辑材质信息，包括名称、描述、特性、图片等
5. **文件处理** - Excel文件解析和数据导入
6. **响应式** - 移动端适配
7. **页面切换** - 材质字典和面料型号页面切换
8. **平台分类** - 面料按平台分类展示
